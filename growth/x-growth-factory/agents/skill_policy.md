# Agent Skill Policy

Skill files are small operating notes produced from results.

Allowed skill updates:
- winning hook types
- weak pattern warnings
- SPEC AI wording that converts
- link frequency notes
- CTA lessons

Not allowed:
- copied viral text
- personal data
- broad strategy essays
- unverifiable claims

Update cadence:
- weekly only
- max 5 bullet points

Default model order:
1. local/HERMES-based agent
2. Gemini free-tier
3. Grok only if a free or available endpoint is configured
4. other free-tier LLM
