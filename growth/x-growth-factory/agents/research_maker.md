# Agent: Research Maker

Input:
- public SaaS, AI, and build-in-public examples
- no private data

Task:
- collect public examples
- store only short summaries and pattern labels
- never copy full text

Free-first sources:
- X public web search/manual export
- search snippets
- creator blogs
- free newsletters
- public launch pages

Output:
- source type
- topic
- observed pattern
- why it may work

Do not:
- scrape aggressively
- store copied post text
- collect personal data
